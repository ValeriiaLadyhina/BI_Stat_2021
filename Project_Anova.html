3. ANOVA analysis 

women_days <- data_drug_effects %>% filter(gender == "Female", drug_type == c('Placebo','NT2')) %>% select(days_in_hospital, drug_type)

Here are the mean values for Days spent in hospital for 2 women groups in the analysis: 

tapply(women_days$days_in_hospital, women_days$drug_type, mean)
##      NT1      NT2 Old Type  Placebo 
##       NA 20.88786       NA 26.19528

Now lets check if there is a statistical difference between these values: 

sstest <- t.test(women_days$days_in_hospital[women_days$drug_type == "Placebo"], women_days$days_in_hospital[women_days$drug_type == "NT2"])
sstest
## 
##  Welch Two Sample t-test
## 
## data:  women_days$days_in_hospital[women_days$drug_type == "Placebo"] and women_days$days_in_hospital[women_days$drug_type == "NT2"]
## t = 4.8295, df = 20.591, p-value = 9.441e-05
## alternative hypothesis: true difference in means is not equal to 0
## 95 percent confidence interval:
##  3.019241 7.595614
## sample estimates:
## mean of x mean of y 
##  26.19528  20.88786

Results

As a result of the Students t-Test analysis, we were able to reject the Null hypothesis that there is no difference for the amount of days in hospital spent by women treated with Placebo vs women treated with Type II drug *(statistic value = 4.8295, women sample size _Placebo = 13, women sample size_Type II drug = 13, p-value = 9.441e-05).
